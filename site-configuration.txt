Modules

* Install and enable menu_block module (https://drupal.org/project/menu_block)
* Patch with http://drupalcode.org/project/menu_block.git/commitdiff/c7c644b

Content

* Create a Home page first, and set this to be used in Site Configuration

Menus:

* Use normal capitalisation; the style sheets take care of all caps/all lower
* Disable secondary links (menu->configuration)

* Main menu contains the links at the top of the page
* Remove/disable the Home link, add other main pages

* Navigiation menu contains the links at the bottom of the page
* Remove/disable existing links
* Add page links in correct heirarchy
* Don't forget to select "show as expanded"
